# 부개고 알리미 · Windows

부개고등학교 게시판 5곳과 홈페이지 팝업을 확인하고, 새 게시물이 있으면 Windows 알림으로 알려주는 데스크톱 앱입니다. 시간표와 급식, 학사일정, 학교소개도 함께 확인할 수 있습니다.

## 사용법

1. `부개고-알리미-Windows.exe`를 실행합니다.
2. 처음 실행할 때는 현재 게시물을 기준점으로 저장하며, 다음 확인부터 새 게시물만 알립니다.
3. `설정`에서 알림 시간, 학년·반, 화면 테마, Windows 자동 시작을 선택합니다.
4. 창의 닫기 단추를 누르면 앱은 종료되지 않고 작업 표시줄의 알림 영역에 남습니다. 완전히 끝내려면 알림 영역 아이콘의 `종료`를 누릅니다.

`학교생활` 탭에서는 원하는 날짜의 급식과 알레르기 번호, 연도·학기별 학사일정 표, 학교알리미의 최신 학교 기본정보를 볼 수 있습니다. 각 화면의 새로고침 또는 조회를 누를 때 원문을 다시 받아오므로 원문 변경사항도 반영됩니다.

초록색은 오늘 게시된 소식, 오늘 학사일정, 현재 진행 중인 교시처럼 현재 시간이나 날짜에 실제로 해당하는 행에만 자동으로 표시됩니다.

새 게시물이 없는 예약 확인에서는 `오늘은 조용하네요.` 알림만 보내며, 이 알림은 앱 기록에 저장하지 않습니다.

알림 시간을 모두 삭제해 0개로 두면 자동 확인과 알림이 꺼집니다. 설정 화면의 `확인하는 소식`에서 각 소식 이름을 누르면 해당 학교 홈페이지가 열립니다.

## 개발 빌드

```powershell
powershell -ExecutionPolicy Bypass -File build_native.ps1
```

Windows 11에 포함된 .NET Framework를 사용하며, 별도의 Python이나 .NET 설치가 필요하지 않습니다. 핵심 소스는 `BugaeNotifier.cs`입니다.

## 데이터 위치

설정과 새 소식 기록은 `%LOCALAPPDATA%\BugaeNotifier`에 저장됩니다.

## 주의

- 시간표, 게시물, 급식, 학사일정, 학교소개 확인에는 인터넷 연결이 필요합니다.
- 학교 사이트나 컴시간의 페이지 형식이 크게 변경되면 수집 기능의 수정이 필요할 수 있습니다.
- 개인 배포용 실행 파일은 코드 서명이 되어 있지 않아 Windows SmartScreen 안내가 표시될 수 있습니다.
